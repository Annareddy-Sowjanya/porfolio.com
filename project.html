Arduino based Gas Leakage Detection System

(B.Tech fnl.yr)

Project Objectives:
  • To analyse the sensor for natural gas leakage detection.
  • To design automatic arduino based device for natural gas leakage detection
  • The device will show an alert message on LCD display and send a msg through GSM along with buzzer activation.

The system comprises an LPG gas leakage detector that communicates and sends the warning signal to the Arduino Uno Microcontroller. The warning signal is then sent via GSM network to alert the specified mobile phone users via SMS.
